Impression

Impression is a social networking website for artists to share their work and find each other. It has a custom user model which allows visitors to cultivate a list of friends and send messages back and forth. It also supports a blog for each individual artist to talk about their process or new pieces.

To find other artists, a visitor could make a specific search or just go to the Gallery page. This page brings together a random sampling of the user base and displays their chosen piece, similar to how other apps have a 'discover' or 'explore' function.

All in all, it was an educational project. Comparing to what Django has to offer, it was pretty small scale. That being said, it was not meant to blow the world away. It was meant for me to get a solid handle on the basics, and I would say that it worked pretty well. After writing Impression over the course of a month, I was able to blast through this website in just a few days.

Snake

If you came all the way here, you might as well have some fun! On the other side of this link is a version of Snake that I wrote using Javascript. Not a lot of bells and whistles, just a nod to a classic.

I've written a couple of minigames like this one using Javascript canvas, which I was inspired to do after a class on computer graphics back in Madison. That class focused mostly on math, vectors, and the graphics pipeline, but I found that I could make something fun with just the basics.

I believe if you inspect the element, you can see the javascript file that runs the game - it doesn't get bundled. That, or it's also on my github.
